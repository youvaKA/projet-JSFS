Règles jeu de Pierre, Feuille, Ciseaux est un jeu de hasard pour deux joueurs. Les joueurs choisissent simultanément l'une des trois options : pierre, feuille ou ciseaux.

    La pierre bat les ciseaux (car elle les casse)

    Les ciseaux battent la feuille (car ils la coupent)

    La feuille bat la pierre (car elle l'enveloppe).

    Si les joueurs choisissent la même option, c'est un match nul.
